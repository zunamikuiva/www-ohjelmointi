# WWW-ohjelmointityöt

Tässä repossa esittelen WWW-ohjelmointiharjoituksiani luentojen ja kotitehtävien pohjalta. Jokainen luento on oma osionsa, ja sisältää koodiesimerkit ja harjoitukset.

---

## Luento 1 – Internet, WWW, Työkalut
- Koodieditoreita, selaimia, VPN, FTP, pääteohjelma  
- Materiaalit ja koodiesimerkit  
`Luento1/`

## Luento 2 – Linkit, kuvat, taulukot ja lomakkeet
- HTML-taulukot ja lomakkeet  
- Kotitehtävä: ilmoittautumislomake  
`Luento2/`

## Luento 3 – CSS ensiaskeleet
- CSS Box Model, lähtötilanne, tunnilla tehty koodi  
`Luento3/`

## Luento 4 – Niisku palvelin, Position absoluten käyttö
- PositionAbsolute, layoutharjoitus, tunnilla tehty koodi  
`Luento4/`

## Luento 5 – Kuvien hyödyntäminen sivutaitossa
- Sivutaitto grafiikalla, CSS-harjoitus  
`Luento5/`

## Luento 6 – Prosentuaalinen mitoitus
- Niiskun käyttöönotto, tiedostonsiirto, Liquid-taittotapa  
`Luento6/`

## Luento 7 – Responsiivinen taittotapa
- Responsive Web Design, tunnilla tehty esimerkki  
`Luento7/`

## Luento 8 – Skeleton framework
- Skeleton framework, yksinkertainen responsiivinen sivupohja

## Luento 9 – Flex layout, Grid layout

## Luento 10 – Bootstrap

`Luento10/`

## Luento 11 – Transform, Transitio, Animaatio

## Luento 12 – CSS + Sass
